UI customization

This sample demonstrates how to customize the events calendar UI.

 

Options used in the sample:

  • titleFormat - the title text format that will be shown in the header bar
  • headerBarVisible - indicates whether the header bar will be visible
  • navigationBarVisible - indicates whether the bottom navigation bar will be visible.
  • statusBarVisible - indicates whether the status bar will be visible
  • rightPaneVisible - indicates whether the right pane will be visible. By default the right pane are empty. You can use this pane in order to further customize the UI.

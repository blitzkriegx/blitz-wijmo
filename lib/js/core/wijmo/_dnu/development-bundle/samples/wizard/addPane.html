The Wijmo Wizard widget (wijwizard) is used to display sections of content on separate, navigable pages so you can save screen real estate or simplify input tasks by creating a series of forms. For easy navigation, you can use wijwizard's built-in navigation buttons or its autoplay feature. And if you'd like to add extra flair to the wijwizard, you can combine it with the wijpager widget for paged navigation.
